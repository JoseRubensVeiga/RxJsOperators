fromEventPattern

Essa função criadora é muito parecida com o fromEvent, porém mais detalhada.

Nela podemos interceptar as funções de attaching e detaching, com isso, conseguimos manipular um evento paralelo para acontecer nestes momentos específicos.

A função de attaching vai ser chamada sempre que houver uma inscrição e a de detaching vai ser chamada sempre que houver uma desinscrição.

Neste exemplo a inscrição começa quando a view é carregada. E a desinscrição é efetuada quando a view é destruída usando o método unsubscribe. Usando pipes limitadores como o take podemos ter o mesmo efeito
